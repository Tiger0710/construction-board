工事予定表の使い方

ブラウザから入力するだけで、大画面に自動で表示されます

このシステムのしくみ

やることはとてもシンプルです。

フォームに入力 → 保存ボタン → 大画面に自動表示

ブラウザの入力フォームから工事の予定を入力して保存するだけで、事務所の50インチ画面に自動で映し出されます。

ポイント: Excelやファイルの操作は不要です。ブラウザだけで完結します。
保存するとすぐに大画面に反映されます。

入力フォームの使い方

下のボタンから入力フォームを開いてください。

入力フォームを開く

  1. フォームを開くと、今月の案件一覧が表示されます
    左右の矢印で前月・翌月に切り替えられます
  2. 「+ 案件追加」ボタンで新しい工事を登録します
    客先、工事件名、担当者、協力会社、開始日、終了日を入力
  3. 案件カードをクリックすると日次入力が開きます
    各日の工事内容、昼/夜、重点工事を入力します
  4. 入力が終わったら画面下の「保存」ボタンを押します
    保存するとすぐに大画面サイネージに反映されます

案件の登録

「+ 案件追加」ボタンを押すと、入力画面が出ます。

入力する項目書く内容
客先発注元の会社名
工事件名工事の名前
現場担当者自社の現場担当者
安品担当者安全品質管理の担当者
協力会社名協力会社の名前
協力会社担当者協力会社の担当者
開始日工事が始まる日
終了日工事が終わる日
大事: 「開始日」と「終了日」は必ず入れてください。日付がないと大画面に表示されません。
編集・削除: 案件カードの右上の「編集」ボタンから、内容の変更や削除ができます。

日次入力

案件カードをクリックすると、開始日から終了日までの日付一覧が開きます。
各日について以下を入力します。

項目入力方法
昼チェックチェックON = 昼の工事あり（デフォルトON）
夜チェックチェックON = 夜の工事あり（デフォルトOFF）
工事内容その日にやる作業を書く
重点工事「重点:有」を選ぶと大画面で目立つ表示になります
重点内容重点工事が「有」のとき、内容を記入（例: 高所作業）
ポイント:
・昼のチェックを外すと、その日は大画面に表示されません（休みの日など）
・夜にチェックを入れると、夜間工事の入力欄が表示されます
・「担当者変更」を押すと、その日だけ現場・安品・協力会社担当者を変更できます
・今日の日付には TODAY マークが付いています

大画面の見方

事務所の50インチ画面には、今日と明日の工事予定が自動で表示されます。

表示のしかた内容
ページ切り替え30秒ごとに自動で切り替わります
重点工事左側にオレンジのラインが付き、重点内容がオレンジバッジに表示されます
夜間工事青い背景・白文字で表示されます
データ更新5分ごとに自動で最新データを取得します
時計右上に現在時刻が出ています

新しい月の準備

月末になったら、入力フォームで翌月のデータを準備します。

  1. 入力フォームを開く
  2. 右矢印ボタンで翌月に切り替え
  3. 「+ 案件追加」で翌月の工事を登録
  4. 日次入力を記入して保存
月またぎ: 前月から翌月にまたがる工事がある場合は、翌月のデータにも同じ案件を登録してください。
前月のデータは翌月初でも表示されるので、急がなくても大丈夫です。

よくある質問

保存したのに大画面に反映されない
大画面は5分ごとにデータを更新します。少し待ってみてください。
間違えて入力した / 消したい
入力フォームで修正して保存すればOKです。案件ごと削除したい場合は、編集ボタンから「削除」を押してください。
他の人が同時に編集したらどうなる？
同時に保存しようとすると「データが更新されています」というメッセージが出ます。
ページをリロードして最新データを取得してから、もう一度入力してください。
スマホからも入力できる？
はい、スマホのブラウザからも入力できます。同じURLにアクセスしてください。
