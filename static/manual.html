1. システム概要

DirectCloud上のExcelファイルに工事予定を入力すると、自動で50インチパネルのサイネージに表示されるシステムです。

Excel入力 → DirectCloud保存 → 自動検出 → HTML生成 → サイネージ表示

ファイル構成

場所内容
E:\★工事共有\工事予定表\2604\2026年4月の入力ファイル
E:\★工事共有\工事予定表\2605\2026年5月の入力ファイル
各フォルダ内入力_担当者名.xlsm (担当者ごとに1ファイル)
ポイント: ファイルは月ごとのフォルダに分かれています。
フォルダ名は YYMM 形式（例: 2604 = 2026年4月）です。

3. 日常の入力作業

毎日の入力作業はこれだけです。

  1. DirectCloudで自分の入力ファイルを開く
    E:\★工事共有\工事予定表\2604\入力_○○.xlsm
  2. ガントチャートシートに工事の基本情報を入力
    客先、工事件名、担当者、協力会社、開始日〜終了日
  3. ファイルを保存 (Ctrl+S)
    → 日次入力シートが自動で生成される (VBA)
  4. 日次入力シートで詳細を記入
    昼/夜、工事内容、重点工事
  5. 再度保存
    → 約30秒で50インチパネルに自動反映
自動同期: 保存するたびにVBAマクロがガントチャートの開始日〜終了日から日次入力シートのエントリーを自動生成します。手入力したデータ（昼/夜、工事内容、重点工事）は上書きされません。

4. 新しい月のファイル作成

月末に翌月のファイルを作成します。

  1. 管理ページを開く
  2. 対象月を入力 (例: 2605 = 2026年5月)
  3. 担当者名を1行ずつ入力
  4. 「ファイル生成」ボタンをクリック
  5. 約2分でDirectCloudにファイルが自動配置される
管理ページ → GitHub Actions → ファイル生成 → DirectCloud配置
生成されるファイル:
E:\★工事共有\工事予定表\2605\入力_田中.xlsm
E:\★工事共有\工事予定表\2605\入力_山本.xlsm
など、担当者ごとに1ファイルずつ作成されます。

5. ガントチャートシートの使い方

工事の基本情報と期間を入力するシートです。

列項目入力内容
A客先発注元の会社名
B工事件名工事のプロジェクト名
C現場担当者自社の現場担当者名
D安品担当者安全品質管理部の担当者名
E協力会社名協力会社の正式名称
F協力会社担当者協力会社の担当者名
G開始工事開始日 (例: 4/10)
H終了工事終了日 (例: 4/15)

I列以降には月の日付が並んでおり、開始〜終了日の期間が青色でバー表示されます。

注意: 開始日と終了日は必ず入力してください。日付がないとサイネージに表示されません。

6. 日次入力シートの使い方

ガントチャートを保存すると、日次入力シートに各日のエントリーが自動生成されます。
以下の項目を手入力で記入してください。

列項目入力方法
A日付自動生成 (編集不要)
B客先自動生成 (編集不要)
C工事件名自動生成 (編集不要)
D昼/夜プルダウンから選択: 昼 / 夜 / なし
E工事内容当日の具体的な作業内容を記入
F重点工事プルダウンから選択: 有 / 無
ポイント:
  • A〜C列はガントチャートから自動で入るので、D〜F列だけ記入すればOK
  • 「なし」を選ぶとサイネージに表示されません（休工日など）
  • 重点工事「有」の行はサイネージで強調表示されます
  • 手入力したデータはガントチャート変更後も保持されます

8. 自動更新モニター (管理者向け)

PC上で常駐するプログラムが、DirectCloudの変更を検出してサイネージを自動更新します。

起動方法

start_watcher.bat をダブルクリック、またはスタートアップに登録済み。

動作

項目内容
監視間隔15秒ごとにファイル変更チェック
git同期30秒ごとにリポジトリから新規ファイル取得
デプロイ変更検出 → 統合 → HTML生成 → Netlify配信
ログlogs\watcher.log
停止ウィンドウで Ctrl+C
注意: モニターが停止していると、ファイルを更新してもサイネージに反映されません。ウィンドウが閉じていないか確認してください。

9. 困ったとき

サイネージに反映されない

  • モニター (start_watcher.bat) が起動しているか確認
  • ファイルが正しいフォルダに保存されているか確認
    E:\★工事共有\工事予定表\YYMM\入力_○○.xlsm
  • ファイル名が 入力_ で始まっているか確認

日次入力シートが自動更新されない

  • Excelでマクロが有効になっているか確認
    (「コンテンツの有効化」ボタンが表示されたらクリック)
  • ガントチャートの開始日・終了日が正しく入力されているか確認

ファイルが開けない / ロックされている

  • 他の人が同じファイルを開いている可能性があります
  • ~$入力_○○.xlsm というファイルがあればロック中

管理ページでファイル生成後、DirectCloudに出てこない

  • モニターが起動していれば約2分で自動配置されます
  • モニターが停止している場合は手動でファイルをコピーしてください
